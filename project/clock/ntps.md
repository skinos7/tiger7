## NTP Server agent management
The management NTP Server agent

#### Configuration( clock@ntps )

```json
// attribute introduction
{
    "status":"start at the system startup"                        // [ disable, enable ]
}
// example for enable the NTP Server agent
{
    "mode":"enable"
}
```
